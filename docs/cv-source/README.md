# CV Source Material

Drop CVs, resumes or notes here so the portfolio copy can be refined from real experience.

Recommended files:

- `cv-ai.pdf` or `cv-ai.md`
- `cv-pqc.pdf` or `cv-pqc.md`
- `experience-notes.md`

Useful information to include:

- Real roles and dates.
- Projects that can be public, private or anonymized.
- Technologies used in each project.
- Measurable outcomes when available.
- Certifications, education and cybersecurity/PQC background.
- Any details that should not be published.
